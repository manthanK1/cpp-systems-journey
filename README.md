# C++ Systems Journey

A public C++ learning journey through **C++ Primer** and **MIT OpenCourseWare (6.096)**, documenting notes, exercises, and **quant-aware systems programming foundations**.

---

## Overview

This repository is my structured public build log for learning modern C++ with a systems-oriented and quantitative development mindset.

The goal is not just to learn syntax, but to develop the kind of thinking required for:

- systems programming
- numerical correctness
- market data handling
- performance-aware engineering
- production-style C++ development

This repo combines:

- **C++ Primer (Stanley B. Lippman, Josée Lajoie, Barbara E. Moo)**
- **MIT OCW 6.096: Introduction to C++**
- hands-on exercises
- concept demos
- quant-aware foundational utilities

## Why This Repo Is Not Just a "Notes Repo"

Most beginner C++ repositories stop at:

- syntax drills
- textbook exercises
- isolated toy programs

This repository goes beyond that by introducing **quant-aware and systems-aware foundational utilities** early in the learning process.

Examples include:

- floating-point precision and epsilon comparisons
- fixed-point price representation (basic)
- rolling mean / variance / covariance
- simple date handling utilities
- CSV price data parsing
- return calculations and basic market-data summaries

This makes the learning path more aligned with:

- quantitative development
- market data engineering
- performance-sensitive programming
- real-world numerical software design
